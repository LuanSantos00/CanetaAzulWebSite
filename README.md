<h1>Caneta Azul Web Site</h1>

### Aplicação desenvolvida para facilitar a organização de amistosos de um time de amigos.

## Tecnologias Utilizadas
* Angular
* Bootstrap  
* Firebase 


## Iniciar o projeto
* Clone o Repositório
* Execute o npm install
* Execute o ng-serve
